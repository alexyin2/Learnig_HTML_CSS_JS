* We'll get started with the basics of how to create a HTML file in mac.

* 在mac中，直接使用文字編輯寫HTML會有編排字體樣式等問題，
* 因此最好的方法是透過Terminal直接在桌面產生一個txt檔案，再將其轉換為html格式。

* 方法：
* 開啟Terminal
* 輸入：
> cd Desktop
> touch name.txt

* You then will get a pure txt file on your desktop.
